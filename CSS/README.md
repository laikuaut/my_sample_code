## CSSの要素

* セレクタ : スタイルを適用する要素
* プロパティ : 適用するスタイルの特性

```css
セレクタ {
    プロパティ: 値
}
```

### セレクタの種類

* ユニバーサルセレクタ(全要素) : \*
* 要素セレクタ : p, divなど
* IDセレクタ : #id
* classセレクタ : .class

### セレクタ使用方法

* a, b : aとbのセレクタそれぞれにスタイルを適用
    + h1, p
* a b : a配下のbセレクタ全てにスタイルを適用
    + div span
* a > b : a直下にあるbだけにスタイルを適用
    + div > span
* a + b : aの次に来るbにスタイルを適用
    + p + p
* ab : a要素のb属性にスタイルを適用
    + p.class1
